# README

## Getting Started

To get started with the Health Insurance Landing Page, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/Chrisavelli/health-insurance-landing-page.git
   cd health-insurance-landing-page
   ```
2. Install the necessary dependencies:
   ```
   npm install
   ```
3. Start the development server:
   ```
   npm run dev
   ```

## Features

- Responsive design that adapts to mobile and desktop devices.
- Intuitive user interface for enhanced user experience.
- Comprehensive TCPA compliance mechanisms.

## Tech Stack

- **Frontend:** React
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **Testing:** Jest

## Deployment

To deploy the project on Vercel, follow these instructions:

1. Sign up for a Vercel account if you don’t have one.
2. Link your GitHub repository to Vercel.
3. Choose your deployment settings and click on the ‘Deploy’ button.

Vercel will build and deploy your project. Changes made to the main branch will automatically trigger a new deployment.
